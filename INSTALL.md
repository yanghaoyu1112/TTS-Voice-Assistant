# TTS 语音助手 - 安装指南

> **适用版本**: MVP v0.9.0  
> **目标读者**: 普通用户（无需编程基础）

---

## 一、系统要求

| 项目 | 要求 | 说明 |
|------|------|------|
| 操作系统 | Windows 10 / 11 | 64位系统 |
| 内存 | ≥ 4GB | 实际占用 < 100MB |
| 显卡 | 无要求 | 纯 CPU 运行，不占用显卡 |
| 网络 | 首次使用建议联网 | 用于下载 edge-tts 语音 |
| 必备组件 | VB-CABLE 虚拟声卡 | 实现语音转发到语音软件 |

---

## 二、安装步骤（推荐）

### Step 1: 安装 VB-CABLE 虚拟声卡

VB-CABLE 是本工具实现"语音转发"的核心组件，它会在系统中创建一个虚拟的麦克风设备，让 Discord / QQ / YY / Teams 等软件可以"听到" TTS 输出。

1. 访问官网下载页面：**https://vb-audio.com/Cable/**
2. 下载 **Windows 安装包**（通常名为 `VBCABLE_Setup_x64.exe`）
3. 双击运行安装程序，按提示完成安装
4. 安装完成后，重启电脑（建议，确保驱动生效）

> 💡 **如何确认安装成功？**  
> 右键点击系统托盘的小喇叭图标 → 打开"声音设置" → 在"输入"设备列表中能看到 **CABLE Output** 或 **VB-Audio Virtual Cable**

---

### Step 2: 下载并运行 TTS 语音助手

#### 方式 A：直接运行 exe（推荐）

1. 获取 `TTS语音助手.exe`（约 60MB 单文件）
2. 将 exe 放到任意目录（如桌面或 `D:\Tools\`）
3. **首次运行双击 exe**

> ⚠️ **杀毒软件提示？**  
> 由于 PyInstaller 打包的单文件 exe 特征，部分杀毒软件（如 360、火绒）可能会误报。请放心添加信任，或联系开发者获取 `--onedir` 目录版作为备选。

4. 运行后，程序会静默驻留在系统托盘右下角（出现一个蓝色麦克风图标）
5. 首次运行会自动创建 `config.json` 配置文件，并存放在 exe 同级目录下

---

#### 方式 B：从源码运行（开发者）

如果你希望修改源码或参与开发，可以按以下步骤搭建环境：

```powershell
# 1. 克隆或解压项目到本地
cd C:\Users\你的用户名\Desktop\tts

# 2. 创建虚拟环境（推荐）
python -m venv venv
venv\Scripts\activate

# 3. 安装依赖
pip install -r requirements.txt

# 4. 开发模式运行
python src/main.py
```

**开发依赖清单**（`requirements.txt` 已包含）：
- PyQt6 ≥ 6.6.0（UI 框架）
- edge-tts ≥ 6.1.0（在线 TTS 引擎）
- pyttsx3 ≥ 2.90（系统 TTS 降级引擎）
- pygame ≥ 2.5.0（音频播放降级）
- sounddevice ≥ 0.4.6（指定设备音频输出）
- soundfile ≥ 0.12.1
- numpy ≥ 1.24.0

---

## 三、打包自己的 exe

如果你想从源码重新打包 exe：

```powershell
# 确保已安装 pyinstaller 和 pillow
pip install pyinstaller pillow

# 执行打包脚本
python build.py
```

打包完成后，exe 会出现在 `dist/TTS语音助手.exe`。

> **打包模式说明**  
> - 默认使用 `--onefile`（单文件，方便分发）  
> - 如遇杀毒软件误报，可修改 `build.py` 使用 `--onedir` 模式降低误报率

---

## 四、安装验证

完成安装后，请检查以下项目：

| 检查项 | 预期结果 |
|--------|----------|
| 双击 exe | 系统托盘出现蓝色麦克风图标 |
| 托盘气泡提示 | 显示"TTS 语音助手已启动" |
| 左键单击托盘图标 | 唤出悬浮输入窗口 |
| 输入"你好"按回车 | 能听到语音播放（从默认扬声器） |

如果以上步骤都正常，说明安装成功。接下来请参考 [QUICKSTART.md](./QUICKSTART.md) 配置语音软件转发。

## 五、常见问题

### Q1: 360 / 火绒提示病毒？

这是 PyInstaller 打包 exe 的常见误报。应对方法：
1. 在杀毒软件中将 `TTS语音助手.exe` 添加为信任
2. 或使用 `--onedir` 模式重新打包（目录版误报率更低）
3. 直接从源码运行，完全避免误报

### Q2: 提示"未检测到 VB-CABLE 虚拟声卡"？

1. 确认是否已安装 VB-CABLE 驱动
2. 确认安装后是否重启过电脑
3. 打开"声音设置"查看输入设备列表中是否有 CABLE 相关设备
4. 若已安装但仍检测不到，可手动在托盘菜单 → 音频设备 中选择正确的输出设备

### Q3: 启动后没有托盘图标？

1. 检查是否被 Windows 通知区域折叠了（点击任务栏右侧向上的小箭头）
2. 查看 exe 同级目录下的 `logs/app.log` 日志文件，排查启动错误
3. 确保系统为 Windows 10/11，且已安装最新的系统更新

### Q4: 程序闪退怎么办？

1. 查看 `logs/app.log` 中的错误信息
2. 确认 Python 版本为 3.12（如从源码运行）
3. 删除 `config.json` 后重新启动，恢复默认配置
