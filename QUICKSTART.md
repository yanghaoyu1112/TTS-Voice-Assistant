# TTS 语音助手 - 快速开始（5 分钟上手）

> 本文档将带你用最快速度完成：启动 → 唤出窗口 → 输入文字 → 让队友听到 TTS 语音。

---

## 第一步：启动程序

双击 `TTS语音助手.exe`，程序会在系统托盘静默运行，右下角出现 **蓝色麦克风图标**。

首次启动时，你会看到一个欢迎气泡：

> "欢迎使用 TTS 语音助手"  
> "首次启动配置已初始化..."

这表示程序已经自动创建了默认配置文件 `config.json`。

---

## 第二步：唤出悬浮输入窗

**方式一（推荐）**：按下全局热键 **`Ctrl + Shift + T`**，即可唤出悬浮窗口。

**方式二**：左键单击或双击托盘图标，同样可以唤出悬浮窗口。

窗口特征：
- 半透明黑色圆角边框
- 可拖动调整位置
- 输入框带蓝色发光边框
- 按 `ESC` 可关闭窗口

> 💡 **小贴士**：窗口首次启动默认显示在屏幕右下角偏上位置，并自动保持在屏幕范围内。你可以用鼠标拖动窗口到任意位置，程序会记忆该位置，下次通过热键或托盘唤出时仍在原地。唤出后输入框会自动获得焦点，可直接输入文字。

---

## 第三步：输入文字并发送

在悬浮窗中输入任意文字，按 **回车键** 发送。

例如输入：

```
前面有敌人
```

你会看到状态栏从"就绪"变为"🔄 正在生成语音..."，随后变为"✅ 播放完成"。

**状态颜色说明**：

| 颜色 | 含义 | 说明 |
|------|------|------|
| 🟢 绿色 | edge-tts | 在线高质量语音（首次稍慢，约 0.5~2 秒） |
| 🟡 金色 | cache | 本地缓存命中（几乎无延迟，< 200ms） |
| 🔴 红色 | sapi5 | 系统 TTS 保底（断网时使用，机械音） |

播放完成后，悬浮窗会在 2 秒后自动隐藏。

---

## 第四步：配置语音软件（关键）

要让 Discord / QQ / YY / Teams 的队友听到 TTS 语音，需要将语音软件的**麦克风输入**设置为 **CABLE Output**（VB-CABLE 的虚拟麦克风）。

### 以 Discord 为例：

1. 打开 Discord → 用户设置 → 语音和视频
2. 找到"输入设备"下拉框
3. 选择 **CABLE Output (VB-Audio Virtual Cable)**
4. 关闭"自动输入灵敏度"，适当调低输入灵敏度（避免环境音干扰）

### 以 QQ / TIM 为例：

1. 打开设置 → 音视频通话
2. 麦克风选择 **CABLE Output**
3. 点击"麦克风测试"，此时说话应无反应，但 TTS 播放时进度条会动

### 以 Windows 系统级设置为例：

如果你希望所有程序默认使用 CABLE：
1. 右键系统托盘小喇叭 → 声音设置
2. "输入" → 选择 **CABLE Output**

> ⚠️ **注意**：设置后你自己的真实麦克风会暂时失效。如需同时说话，可考虑使用 VoiceMeeter 等混音软件（超出本工具范围）。

---

## 第五步：托盘菜单功能一览

右键点击托盘图标，打开功能菜单：

| 菜单项 | 功能 |
|--------|------|
| 🎯 显示悬浮窗 | 手动唤出输入窗口 |
| 🔊 音频设备 | 查看/切换音频输出设备（支持自动检测 VB-CABLE） |
| 🎚️ 音量 | 调节 TTS 播放音量（25% / 50% / 75% / 100%） |
| ⏹ 停止播放 | 立即停止当前播放并清空队列 |
| 📊 缓存统计 | 查看本地缓存文件数量和大小 |
| 🗑️ 清空缓存 | 删除所有缓存的 MP3 文件 |
| ❌ 退出 | 完全退出程序 |

**音频设备切换**：
- 默认是"自动检测"，程序启动时会自动寻找名称中包含 "CABLE" 的设备
- 如果自动检测失败，可以手动在这里选择正确的输出设备
- 选择后会自动保存到 `config.json`，下次启动依然生效

## 常用技巧

### 1. 常用语秒发

程序启动时会后台预缓存以下常用语：

```
救命、支援、撤退、谢谢、抱歉、你好、收到
```

这些词语在第一次播放后会存入本地缓存，之后再次发送时几乎零延迟。

### 2. 离线也能用

- **缓存命中**：完全不需要网络，离线秒播
- **新文本 + 有网络**：使用 edge-tts 在线生成（高质量）
- **新文本 + 断网**：5 秒超时后自动降级为系统 TTS（机械音，但保证能用）

### 3. 连续发送不丢包

快速发送多条文本时，程序会自动排队播放。如果想打断当前播放立即发新消息，只需在悬浮窗中输入新文字并回车，当前播放会被中断，立即播放最新内容。

### 4. 文本长度限制

单次最多支持 **500 个汉字**，超出会自动截断，避免生成失败。

## 故障速查

| 现象 | 可能原因 | 解决办法 |
|------|----------|----------|
| 听不到声音 | 音频设备选成了扬声器而非 CABLE | 托盘菜单 → 音频设备 → 选择 CABLE Input |
| 队友听不到 | 语音软件麦克风没设成 CABLE Output | 按第四步配置语音软件 |
| 首次播放慢 | edge-tts 在线生成需要时间 | 正常现象，第二次同一文字会命中缓存 |
| 断网时音质差 | 降级到了系统 TTS | 正常现象，恢复网络后自动恢复高质量 |
| 悬浮窗不消失 | 播放失败 | 查看状态栏提示，3 秒后会自动恢复 |
